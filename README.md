# yuzu.com — Sistem Dual Website & Telegram Bot

**Versi:** 1.0 (Baseline)  
**Status:** Kerangka dasar siap. Data masih dummy (contoh). Selanjutnya akan diintegrasikan dengan Firebase.

---

## 📌 Tentang Proyek
Proyek ini adalah pusat kendali Telegram (Backend Node.js) yang terhubung dengan dua website dinamis:
1. **Mini App Publik** (`app/docs/`) — Berjalan di dalam Telegram untuk pengguna umum (upload file).
2. **Dashboard Admin** (`monitor/`) — Berjalan di browser biasa untuk memonitor aktivitas bot secara real-time.
